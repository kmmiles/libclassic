Const baseSpellCrit

baseSpellCrit: 1.8 = 1.8
  • Defined in common.ts:42

At level 60, caster classes have some expected amount of Int that will put them at 5% spell crit. For example, to have 5% crit at 60 a mage needs 286 Int. A 60 mage also needs 59.5 int to gain 1 additional spell crit. 286/59.5=4.8067 which is less than 5, meaning mages have a base spell crit of 5-(286/59.5)=0.1933. Likewise, a Shaman needs 160 int @ 60 for 5% crit, and 59.2 int for 1 crit. 160/59.2=2.703 -> 5-(160/59.2)=2.2973 base spell crit
